Tab 9 — Agent/Office Info

Fill Agent/Office Info

Universal — reads agent/office data from clipboard payload. Builder-specific values (Type, Limited Rep) are resolved by the session before the payload is generated.

Fill Agent/Office Info
To install:
1. Make bookmarks bar visible — Cmd+Shift+B
2. Drag the blue button to the bookmarks bar
3. Place in the Matrix — New, Matrix — Tax ID, or Matrix — Copy folder (this tab behaves the same on all three paths)

To use:
1. Copy the full JSON payload to clipboard
2. Navigate to the Agent/Office Info tab in Matrix
3. Click the bookmark — fields populate instantly
4. List Agent Code (Input_159) is pre-filled by Matrix — this bookmarklet never touches it
Payload reference — agent_office key
"agent_office": { "co_list_agent_code": "", // blank if no co-agent; from agent profile "type": "ER", // "ER" (Exclusive Right) standard, "MO" (MLS Only) for Lennar "limited_rep": "0" // "0" (No) standard, "1" (Yes) for Lennar }

Resolved by the session per builder before the payload is generated — see Lennar_Bookmarklet_Customization.md for Lennar values, or New_Seller_Side_Session_Protocol.md for the standard default.
