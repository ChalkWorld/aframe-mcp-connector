To install:
1. Make bookmarks bar visible — Cmd+Shift+B
2. Drag the blue button to the bookmarks bar
3. Place in the Matrix — New, Matrix — Tax ID, or Matrix — Copy folder

To use:
1. Copy the full JSON payload to clipboard
2. Navigate to the Remarks tab in Matrix
3. Click the bookmark — remarks populate from clipboard; Copyright Agreement set to Yes automatically
⚠️ Character limits: Public Remarks 2048 chars, Agent Comments 1000 chars. Enforced client-side. Verify length before saving if remarks are unusually long.

⚠️ Field ID note: this tab uses Input_107 / Input_108 / Input_662, confirmed against the live launcher file. A separate doc (Lennar_Features_Payload_Schema.md) references different IDs (Input_45/Input_46/Input_47) for the same fields — that's a stale cross-reference in that doc, not a discrepancy in this bookmarklet.
