Lennar Features Bookmarklet

CVRMLS Matrix · Features Tab · Lennar new construction only · All chunks in one pass
Lennar only · Single bookmarklet

Install

  1. Show your bookmarks bar — Cmd+Shift+B (Mac) or Ctrl+Shift+B (Windows)
  2. Drag the button below to your bookmarks bar
  3. To use: copy the Lennar Features payload (must include community key), navigate to the Features tab in Matrix, click the bookmark
  4. Statics fire automatically — only dynamic fields come from the payload

Payload — features key + community key

The community key drives the lookup table (heating, heat/fuel, pool, amenities). All other dynamic fields come from features. Community options: Harpers Mill TH, Harpers Mill SF, Creekside Run, Everstone, Watermark.
